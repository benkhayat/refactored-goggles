# refactored-goggles
JUDGES BY THE NUMBERS

For **conference** records, go to https://uaasports.info/ , find the sport you want to format, and in the records table copy EVERYTHING BESIDES THE TITLES OF THE COLUMNS. Paste that into records.txt, and run **UAAFormatter.**

For **OVERALL** records, go to https://uaasports.info/ , find the sport you want to format, and in the records table copy EVERYTHING BESIDES THE TITLES OF THE COLUMNS. Paste that into records.txt, and run **overallFormatter.**
